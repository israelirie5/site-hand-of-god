# Arborescence photos Hand of God

Depose les images finales dans ces dossiers. Garde des noms simples, sans espaces, par exemple `hero-01.jpg`, `distribution-02.webp`.

## Accueil

- `home/hero` : grandes images du slider hero.
- `home/impact` : images liees aux chiffres d'impact.
- `home/about` : image de presentation de l'ONG.
- `home/actions` : images utilisees dans le bloc "Nos actions".
- `home/gallery-preview` : images mises en avant sur l'accueil.
- `home/testimonials` : portraits ou photos d'ambiance pour temoignages.

## Actions

- `actions/orphelinat-yopougon-mami-adjoua`

- `actions/douce-mere-azuretti-annee-suivante`
- `actions/maison-du-potier-yopougon-niangon-nord`

Chaque dossier represente un album d'action.

## Projets

- `projects/prochain-projet` : visuels de teasing du prochain projet.
- `projects/_nouveau-projet-template` : duplique ce dossier quand un nouveau projet est cree.

## Articles

- `articles/orphelinat-yopougon-mami-adjoua`

## Galerie

- `gallery/albums` : albums generaux. La galerie du site est pensee pour afficher les albums d'actions automatiquement.
